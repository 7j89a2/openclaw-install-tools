2. In the main menu, choose **LAN P2P Transfer**.
3. The app will search for other devices running this tool on your network.
4. Select the device to connect.
5. Once connected, drag and drop files into the interface window.
6. The transfer runs automatically and shows progress.
7. When finished, files will be saved in the chosen destination folder.
